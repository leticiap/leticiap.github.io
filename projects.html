Game Jam Projects

Cat Vibes

This is the game about a demon cat working for the evil Cathomet! Your objective is to destroy all the hapiness and good vibes from the happy Dog Village. This game was developed with Godot and Asesprite.


Platform: HTML5
Year: 2018
Commands: Use the arrow keys to move you character. Use the spacebar to collect and transmit bad vibes.

Extinction Run

The humans destroyed Roni's home. Seeking for revenge, the Rhino sets off on a race to destroy the humans. This infinity run is made with Unity.


Platform: PC

Commands: Use the arrow keys to avoid the obstacles. Use 'A' and 'S' to attack and destroy objects.

Unity Games

Galaxy Shooter

A basic galaxy shooter game from the course 'The Ultimate Guide To Game Development With Unity'. The game has single and co-op modes, as well as a score system, power-ups and some nice effects with post processing.


Platform: PC
Commands: Use the left mouse button (player 1) or the space bar (player 2) to shoot, and the arrow keys (player 1) or WASD keys (player 2) to move.
Download the game - source code


Sci-fi Demo

A 3D first person controller game from the course 'The Ultimate Guide To Game Development With Unity'. It implements some 3D concepts as collision detection, raycast, and also other techniques such as animations and post processing.


Platform: PC
Commands: Use your mouse to aim and rotate the camera, and the arrow keys to move. To shoot, use the left mouse button.
Download the game - source code


C++ Games

Timber!

A frantic two-button tapper from the course 'Learning Path: C++ Game programming'. Based on the classic Timber game, your objective is to chop down the tree. It is implemented with SFML, a library to simplify OpenGL.


Platform: PC
Commands: Use the left and right arrow keys to chop the tree.
Download the game - source code



Zombie Arena

A multi-level zombie survival shooter from the course 'Learning Path: C++ Game programming'. Your objective is to kill all the zombies from the level, and you can choose a boost before each level. It is also implemented with SFML.


Platform: PC
Commands: Use the mouse to aim and WASD keys to move. To shoot, use the left mouse button, and to recharge, R button.
Download the game - source code


Thomas Was Lost

A multiplayer puzzle-platformer from the course 'Learning Path: C++ Game programming'. Based on the puzzle game Thomas Was Alone, your objective is to take the two characters to the goal. It is also implemented with SFML, and it has a particle system.


Platform: PC
Commands: Use the left and right arrow keys to chop the tree.
Download the game - source code
